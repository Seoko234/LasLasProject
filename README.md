# FLHP

Paper 1.21.5용 마인크래프트 자석석 집/벌금 플러그인입니다.

## 기능

- 플레이어가 자석석을 설치하면 그 자석석 위치가 자신의 집으로 등록됩니다.
- 등록된 자석석 위에서 서버를 나가면 벌금이 붙지 않습니다.
- 등록된 자석석 위가 아닌 곳에서 서버를 나가면 다이아몬드 5개 벌금이 붙습니다.
- 벌금을 내지 않은 플레이어는 접속 중 느림 효과를 받습니다.
- 벌금을 내면 느림 효과가 바로 사라집니다.
- 등록된 자석석 집 위 정중앙에 빨간 파티클이 표시됩니다.

## 명령어

- `/벌금 확인` - 현재 벌금을 확인합니다.
- `/벌금 지불` - 다이아몬드로 벌금을 납부합니다.
- `/fine 확인`, `/fine 지불` - 영어 별칭 명령어입니다.
- `/flhp 확인`, `/flhp 지불` - 플러그인 이름 별칭 명령어입니다.

## 빌드 방법

```bash
mvn package
```

빌드가 끝나면 아래 위치에 플러그인 JAR 파일이 생성됩니다.

```text
target/FLHP-1.0-SNAPSHOT.jar
```

## 서버 적용 방법

Paper 1.21.5 서버의 `plugins` 폴더에 빌드된 JAR 파일을 넣고 서버를 재시작하면 됩니다.

예전 버전의 JAR 파일이 `plugins` 폴더에 남아 있으면 충돌하거나 헷갈릴 수 있으니, `DisasterPlugin`, `FengLasHomePlugin` 같은 예전 이름의 JAR은 제거하고 `FLHP-1.0-SNAPSHOT.jar`만 넣는 것을 추천합니다.
